# Typing Speed Test 

Μια διασκεδαστική και διαδραστική εφαρμογή που μετράει την ταχύτητα και την ακρίβεια δακτυλογράφησης του χρήστη σε πραγματικό χρόνο! Το project δημιουργήθηκε με την Python και το σύγχρονο framework Flet για την εξάσκηση σε χρονομετρήσεις και επεξεργασία κειμένου.

## Τι ακριβώς κάνει;
* **Τυχαίες Προτάσεις:** Επιλέγει τυχαία μια πρόταση από μια μεγάλη λίστα κάθε φορά που πατάς "Ξεκίνημα".
* **Αυτόματο Χρονόμετρο:** Καταγράφει ακριβώς τη χρονική στιγμή έναρξης και λήξης χωρίς να το βλέπει ο χρήστης για απόλυτη ακρίβεια.
* **Υπολογισμός WPM & Ακρίβειας:** Υπολογίζει τις Λέξεις Ανά Λεπτό (Words Per Minute) και συγκρίνει το κείμενο του χρήστη με το αρχικό κείμενο γράμμα-προς-γράμμα για να βγάλει το ποσοστό ακρίβειας (%).
* **Δυναμικά Χρώματα:** Το τελικό αποτέλεσμα αλλάζει χρώμα ανάλογα με την απόδοση (Πράσινο για 100%, Κίτρινο για >80%, Κόκκινο για τα υπόλοιπα).


**Βασικές προγραμματιστικές τεχνικές:**
1. **Η βιβλιοθήκη `time`:** Χρήση της `time.time()` για τον υπολογισμό της διάρκειας εκτέλεσης μιας ενέργειας.
2. **Επεξεργασία Συμβολοσειρών (Strings):** Χρήση της μεθόδου `.split()` για τον υπολογισμό των λέξεων και της συνάρτησης `zip()` για την ταυτόχρονη σύγκριση δύο κειμένων.
3. **Βελτίωση User Experience (UX):** Ενεργοποίηση/Απενεργοποίηση πεδίων (`disabled=True/False`) και αυτόματη εστίαση (`input_text.focus()`) ώστε ο χρήστης να ξεκινάει να γράφει αμέσως.
4. **Η βιβλιοθήκη `random`:** Χρήση της `random.choice()` για την επιλογή τυχαίων στοιχείων από μια λίστα.
